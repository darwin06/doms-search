== Description ==
Plugin to enable filter search by PDF's, Doc's, Images and/or Videos

== Installation ==
- Goto **Plugins** select click **Add new**
- Select the plugin file **doms-search.zip**
- Click **Install Now**
- Goto **Settings > DOMS Search Menu**
- Firstable select the template between Tag *SELECT* or *CHECKBOX*
- Next enable/disable the filters (PDF, DOC, IMG, VIDEO) to show in the Search page results.
- Save the changes.
- Goto your search page results e.g. https://your-custom-domain.com/?s=hello+world
